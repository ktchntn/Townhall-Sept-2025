Welcome to the 90s Web Experience!

This project showcases various nostalgic 90s-style web components including popups, banners, and glitter effects.

Available Components:

  • Nineties Popup - Authentic 90s-style popup windows with draggable functionality
  • Vertical Ad Banner - Classic sidebar advertisements with blinking text
  • Glitter Background - Add sparkles and glitter effects to any element
